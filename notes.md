### git commands
`git status`: lists files in the repository that have not yet had their changes committed
`git add "File"`: adds the named file to the repository
`git commit -am "(update message)"`: commits any changes in the repository, using the update message as a note
`git push`: uploads repository from the environment to the cloud
`git pull`: downloads repository from the cloud to the environment
